##  Overview
This project focuses on designing the UI/UX of a TikTok-like social media
application using Figma, aiming to enhance user engagement and usability.

##  Objectives
- Design intuitive user interface for short-video social platform
- Improve user experience through clear navigation and interaction flow
- Apply UI/UX principles and design system

##  Tools & Technologies
- Design Tool: Figma
- Design Approach: User-centered design
- Platform: Mobile Application

##  User Flow
- Browse videos (For You)
- Like, comment, share videos, livestreams
- Upload short videos
- Purchase and payment
- Manage personal profile

##  Design Highlights
- Minimal and modern UI
- Consistent color and typography
- Bottom navigation optimized for one-hand usage
